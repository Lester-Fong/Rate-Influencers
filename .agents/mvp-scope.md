# MVP Scope

Build only these features:

## Public
- Home page influencer grid
- Search influencers
- Influencer detail page
- Read approved reviews
- Submit review as pending

## Admin
- Login/logout
- Dashboard placeholder
- Influencer CRUD
- Review moderation: approve/reject

## Not part of MVP
Do not build:
- helpful votes
- reports
- advanced analytics
- rating dimensions
- user accounts
- influencer claiming
- notifications
- audit logs
- complex roles
- recommendation system
